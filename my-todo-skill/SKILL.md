# My To-Do Skill
This skill allows you to manage your to-do list by adding, viewing, and removing tasks. You can interact with the skill using natural language commands.

## Features
- Add a task to your to-do list.
- View all tasks in your to-do list.
- Remove a task from your to-do list.

## Usage

### Adding a Task
To add a task, simply say:
```Add [task name] to my to-do list.
For example:
Add "Buy groceries" to my to-do list.
```

### Viewing Tasks
To view all tasks in your to-do list, say:
```
Show me my to-do list.
This will display all the tasks you have added. 

```

## Conclusion
The My To-Do Skill is a simple and efficient way to manage your tasks. You can easily add, view, and remove tasks using natural language commands. Start organizing your tasks today with My To-Do Skill!
