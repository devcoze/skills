# My To-Do Skill
This skill allows you to manage your to-do list by adding, viewing, and removing tasks. You can interact with the skill using natural language commands.

## Features
- Add a task to your to-do list.
- View all tasks in your to-do list.
- Remove a task from your to-do list.

## Usage
### Adding a Task
To add a task, simply say:
```Add [task name] to my to-do list.
```For example:
```Add "Buy groceries" to my to-do list.
````
